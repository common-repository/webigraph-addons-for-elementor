== Installation ==
Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.


1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress.
3. Now you can go the elementor editor section and check there have webigraph addons. There have Webigraph Addons elementors. 
4. Then You can drop and drag this elements.
5. Create inline multi buttons and texts or headings with easy to customizations. 
6. Done!
